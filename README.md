Meteor Trianglify
==================

[Trianglify.js](https://github.com/qrohlf/trianglify) packaged for [Meteor](http://meteor.com).


### Installation

With [Meteorite](https://github.com/oortcloud/meteorite) installed:

```sh
$ mrt add trianglify
```

### Usage

See [here](https://github.com/qrohlf/trianglify)
